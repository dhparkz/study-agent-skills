# 참고 C: 완전한 스킬 예시(Complete Skill Examples)

이 가이드의 패턴을 보여주는 프로덕션 레벨 스킬 예시:
- Document Skills — [PDF](https://github.com/anthropics/skills/tree/main/skills/pdf), [DOCX](https://github.com/anthropics/skills/tree/main/skills/docx), [PPTX](https://github.com/anthropics/skills/tree/main/skills/pptx), [XLSX](https://github.com/anthropics/skills/tree/main/skills/xlsx) 생성
- [Example Skills](https://github.com/anthropics/skills/tree/main/skills) — 다양한 워크플로 패턴
- [Partner Skills Directory](https://www.claude.com/connectors) — Asana, Atlassian, Canva, Figma, Sentry, Zapier 등 다양한 파트너의 스킬을 볼 수 있습니다.

이 레포들은 최신 상태로 유지되며, 가이드에서 다루지 않은 추가 예시도 포함합니다. 클론해서 목적에 맞게 수정하고 템플릿으로 활용하세요.
